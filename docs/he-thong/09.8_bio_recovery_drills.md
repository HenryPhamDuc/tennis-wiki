# 🧬 Bio-Recovery Drills — Hồi phục sinh học

## Daily Protocol: Neuro-Reset 16s

**Mục tiêu**: Reset $\sigma_{sync}$ giữa các điểm

| 0-4s         | 4-12s           | 12-16s                |
| ------------ | --------------- | --------------------- |
| Nhìn dây vợt | Hít 4s + thở 8s | Visualization cú tiếp |

## Drill 1: Plyometrics (HRV Xanh)

**Mục tiêu**: Tăng SSC + độ dày mạc

**Thực hiện**:

- Box jumps
- Depth jumps
- Lateral bounds
- 3-4 sets, 6-8 reps

## Drill 2: Dynamic Stretching

**Mục tiêu**: Đánh thức hệ mạc (KHÔNG static)

- Leg swings
- Arm circles
- Hip openers
- Thoracic rotations

## Drill 3: Fascial Self-Massage

**Mục tiêu**: Mạc đàn hồi

**Dụng cụ**: Foam roller, lacrosse ball

**Vùng**:

- IT band
- Glutes
- Thoracic spine
- Plantar fascia

## Drill 4: Vagus Nerve Activation

**Mục tiêu**: Hạ nhịp tim nhanh

**Thực hiện**:

- Hít 4s, thở 8s (thở dài hơn hít)
- 4-6 chu kỳ
- → Kích hoạt phế vị → $\sigma_{sync}$ reset

## Drill 5: HRV-Based Weekly Cycle

| HRV  | Loại tập                      |
| ---- | ----------------------------- |
| Xanh | Explosive (Sprints, Plyo, VE) |
| Vàng | Technical (Shadow, AI)        |
| Đỏ   | Recovery (Yoga, mạc)          |

## Drill 6: Hydration + Electrolyte

- KHÔNG nước lọc quá nhiều
- Mg, K, Na electrolytes
- Collagen + Vitamin C trước tập 30-60'
