# 🌌 GSM (Gravitational Shift Multiplier) — Trợ lực trọng trường

## Concept

**Stop "pushing" with muscles; start "falling" into your movements.**

Split-step là **gravitational drop** — bằng cách hạ trọng tâm, bạn nhận năng lượng "miễn phí" để bùng nổ lateral.

## Công thức mượn lực Trái đất

$$S_{gravity} = 1 + \alpha \left( \frac{h_{player}}{h_{drop}} \right)$$

| Biến         | Ý nghĩa                                 |
| ------------ | --------------------------------------- |
| $h_{drop}$   | Độ rơi trọng tâm trong Split-step       |
| $h_{player}$ | Chiều cao cơ thể                        |
| $\alpha$     | Hệ số chuyển hóa (khả năng fascia chân) |

**Kết luận**: Nếu nhảy lên + thả rơi sâu → **+15-20% tốc độ bước chạy đầu tiên** mà không tốn sức.

## Stretch-Shortening Cycle (SSC)

Khi trọng tâm rơi (Sinking):

1. Gân gót (Achilles) + mạc đùi bị kéo dãn
2. Tích lũy thế năng
3. Khi đạp đất → năng lượng phóng thích tức thì

→ Lý do tay vợt giỏi trông như "lướt" trên sân.

## Sinking Triết lý

| Sai lầm                            | Đúng                                |
| ---------------------------------- | ----------------------------------- |
| Nhấc cao gót chân, đẩy bằng cơ đùi | "Thả rơi" trọng tâm, mượn trọng lực |
| Mỏi cơ nhanh, mất thăng bằng       | Bền bỉ, ít tốn sức                  |
