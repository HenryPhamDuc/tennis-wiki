# 🎯 DL (Direct Load) — Trả giao bóng nạp lực trực tiếp

## Concept

Biến chính năng lượng của đối thủ thành vũ khí phản công. **Loại bỏ các chuyển động thừa**.

## Vấn đề: Nghịch lý Backswing

| Sai lầm              | Hệ quả                                        |
| -------------------- | --------------------------------------------- |
| Vung vợt ra sau rộng | Tốn thời gian → trễ điểm chạm → bóng ra ngoài |
| **DL Solution**      | Nén thân, mặt vợt luôn sẵn sàng               |

## Phương trình thời gian

$$t_{tổng} = t_{nhận diện} + t_{nạp lực(DL)} + t_{tiếp xúc} < t_{bóng bay}$$

| Biến                   | Giá trị             |
| ---------------------- | ------------------- |
| $t_{nhận diện}$        | ~0.15s              |
| **$t_{nạp lực (DL)}$** | **Biến tối ưu hóa** |

## Cơ chế DL

### 1. Split-step: Lò xo nạp năng lượng

$$J_{load} = m_{player} \times v_{drop}$$

- Nhảy nhẹ lên + "thả rơi" trọng tâm
- Nén fascia ở bắp chân + đùi
- Chạm đất **đúng lúc đối thủ chạm bóng**

### 2. Torso Compression (Nén thân)

- Xoay vai góc nhỏ: **30°-45°**
- Đồng thời dồn trọng tâm sang chân trụ
- Tạo căng thẳng đàn hồi ở **Obliques + ngực**

## Điểm chặn tối ưu (Interception)

$$Tọa độ chặn = (v_{bóng} \times t_{bay}) - (v_{người} \times t_{chạy})$$

**Chiến thuật "Sát vạch"** (Inside the court):

- Dâng cao 1m → giảm thời gian phản ứng đối thủ
- Yêu cầu **NKF cực cao**

## K_wrist trong DL

$$K_{wrist} \geq 0.95$$

Mặt vợt phải là "bức tường" — **mượn chính lực đối thủ** để phản đòn.

## Tránh Over-swing

**Nguyên tắc DL**: Vung ngắn, chạm chắc, dùng hông đẩy.
