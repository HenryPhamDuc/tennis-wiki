### Cleanup strategy

**4-pass regex strip + 2 cleanup passes** because the section structure varied across files:

1. **Pass 1 — section strip**: Remove the `## Nội Dung Gốc (Tiếng Anh)` heading + the opening ` ``` ` fence line (regex matched to first ` ``` ` line — non-greedy).
2. **Pass 2 — orphan OCR**: 238 files had OCR content orphaned after the heading + opening fence were eaten but before the closing fence. Located each file's stray single closing fence, walked back to the last introductory `---` separator, and deleted the orphan OCR + closing fence.
3. **Pass 3 — orphan OCR after intro separator**: 60 files where the OCR content sat between the `---` separator and the next `## ` heading. Detected via Vietnamese-diacritic density heuristic (low VI char ratio = OCR, not legitimate VI summary).
4. **Pass 4 — final residual**: 6 files with substantive copyrighted content (Springer 2009, ISBN 1-84126-075-4, Tennis Australia) that didn't have `**Chủ đề chính:**` markers. Stripped via copyright-marker detection (ISBN, Springer-Verlag, All rights reserved, etc.).
