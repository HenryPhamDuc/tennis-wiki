# Tennis Knowledgebase Hub

> **A network of tennis concept nodes** inspired by the legendary tennisplayer.net teaching library. Each node is a single tennis concept with an original summary, cross-linked to related concepts. **No verbatim text is used**—all content is original synthesis.

## What is a Knowledgebase Hub?

Unlike a traditional article-based wiki, this knowledgebase organizes tennis knowledge as a **graph of concept nodes**. Each node represents one concept (e.g., "Split Step", "Kinetic Chain", "Percentage Tennis"), with:

- A short **original summary** (200-400 words)
- **Cross-links** to 3-7 related concept nodes
- **Source inspiration** (the original tennisplayer.net article that inspired the node)

This structure helps you:

- **Explore** a concept and its relationships
- **Compare** related concepts side by side
- **Learn** through contextual cross-references
- **Avoid copyright issues** by reading original synthesis, not source articles

1. **Start with the Fundamentals** — concepts like "Kinetic Chain" and "Split Step" are foundational to everything else
2. **Follow cross-links** — each node links to 3-7 related concepts to build a deeper understanding
3. **Browse by category** — the structure above groups concepts by domain (biomechanics, tactics, mental game)
4. **Use for reference** — when you encounter a tennis term, search for it here
