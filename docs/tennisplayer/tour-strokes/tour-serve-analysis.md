# Tour Serve Analysis

Modern ATP serves feature extreme variety: 130+ mph first serves, heavy kick second serves, and precise slice serves. Players like Isner, Opelka, and Karlovic use height to generate angle. Others like Federer and Djokovic use variety and placement. The biomechanics involve the full kinetic chain plus extreme internal shoulder rotation.
