# Group Lesson Games

Group lesson games maximize engagement and touches per player in a multi-player setting. Common formats: round-robin rotations, elimination tournaments, target challenges, and team-based competitions. Effective group games keep all players active simultaneously, building fitness, social engagement, and stroke consistency.
