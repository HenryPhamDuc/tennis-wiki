# Core Strength for Tennis

Core strength is fundamental to all tennis strokes and movements. The core transfers power from the lower body to the upper body and stabilizes the spine during rotation. Effective core training includes rotational movements (medicine ball throws, cable rotations), anti-rotation exercises (planks, pallof press), and dynamic stability work.
