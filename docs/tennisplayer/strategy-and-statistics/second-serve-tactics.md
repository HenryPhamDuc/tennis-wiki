# Second Serve Tactics

The second serve is a critical shot in modern tennis, often underrated by amateur players. Tactics: prioritize consistency and depth over pace, use spin to create high bounce (kick serve), and place the serve to set up the third shot. Many amateur points are lost on double faults, so reliability trumps power.
