# The One-Handed Backhand Slice

The slice backhand is a strategic tool that uses underspin to keep the ball low, neutralize pace, and buy recovery time. Unlike topspin groundstrokes that require full body rotation, the slice can be hit with a more compact motion, making it useful in defensive situations and as a setup shot. Champions like Federer demonstrated that the slice remains elite when used with precision and disguise.
